### Step 1: Dry-run 10 profs

```bash
unset http_proxy https_proxy HTTP_PROXY HTTPS_PROXY all_proxy
DATABASE_URL=postgresql+psycopg://miroflow:miroflow@localhost:15432/miroflow_real \
  uv run python scripts/run_homepage_paper_ingest.py --dry-run --limit 10 \
  > /tmp/w9-5-dry-run.log 2>&1
```

记录关键数字：profs_processed / profs_skipped / papers_linked_total (would-be) / full_text_fetched_total / pipeline_issues_filed。

### Step 2: Wet-run 5 profs（真实写入）

挑 dry-run 干净的 5 个；如不好挑用默认 limit 5。

**Before SQL**:
```sql
SELECT count(*) FROM professor_paper_link WHERE evidence_source_type='personal_homepage';
SELECT count(*), source FROM paper_full_text GROUP BY source;
SELECT count(*), match_source FROM paper_title_resolution_cache GROUP BY match_source;
```

**Run**:
```bash
DATABASE_URL=postgresql+psycopg://miroflow:miroflow@localhost:15432/miroflow_real \
  uv run python scripts/run_homepage_paper_ingest.py --limit 5 \
  > /tmp/w9-5-wet-run.log 2>&1
```

记 run_id。

**After SQL**: 同 Before；额外加：
```sql
SELECT issue_type, count(*) FROM pipeline_issue WHERE run_id = '<this-run-id>' GROUP BY issue_type;
```
