# 应用模块

| 应用 | 说明 |
|------|------|
| [miroflow-agent](miroflow-agent/index.md) | 核心 Agent 框架（主应用） |
| [collect-trace](collect-trace/index.md) | 训练数据收集与格式转换 |
| [gradio-demo](gradio-demo/index.md) | Gradio Web UI 演示界面 |
| [visualize-trace](visualize-trace/index.md) | Flask 追踪分析仪表板 |
| [lobehub-compatibility](lobehub-compatibility/index.md) | LobeChat 集成适配器 |
